# Git Version Control Demo

## Objective
Demonstrate basic Git operations including repository creation, branching, merging, and pushing code to GitHub.

## Git Commands Used

```bash
git init
git status
git add .
git commit -m "Initial Commit"
git branch feature
git checkout feature
git merge feature
git remote add origin <repository-url>
git push -u origin main
git pull origin main
```

## Features
- Git Initialization
- Commit Changes
- Branching
- Merge
- Push to GitHub
- Pull from GitHub
