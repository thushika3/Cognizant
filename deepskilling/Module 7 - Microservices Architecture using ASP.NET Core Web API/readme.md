# JWT Authentication in ASP.NET Core Web API

## Description
This project demonstrates JWT Authentication in ASP.NET Core Web API using .NET 8.

## Features
- User Login
- JWT Token Generation
- Protected API using [Authorize]
- ASP.NET Core Web API
- .NET 8

## Technologies
- ASP.NET Core 8
- C#
- JWT Bearer Authentication
- Visual Studio 2022

## Endpoints

POST /api/Auth/login
GET /api/Product (Requires JWT Token)
